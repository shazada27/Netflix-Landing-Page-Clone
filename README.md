 Netflix Landing Page Clone

📌 Project Overview
A visually accurate, front-end web clone of the Netflix homepage. This project focuses on structuring complex web layouts, managing media elements, and building a clean, user-friendly interface using modern web development practices.

 🚀 Technical Highlights
*   **Semantic Structure:** Utilizes semantic HTML elements such as `<nav>`, `<section>`, and `<footer>` for clear and accessible page organization[cite: 4].
*   **Media Integration:** Embeds auto-playing, looped, and muted `<video>` elements alongside standard images to accurately replicate Netflix's dynamic TV and device display mockups.
*   **Structured FAQ Layout:** Features a comprehensive Frequently Asked Questions (FAQ) section, utilizing inline SVG styling for the interactive visual indicators. 
*   **Navigation & Footer Design:** Includes a structured top navigation bar featuring language selection and sign-in buttons, paired with a complete multi-column grid layout for the footer links.
*   **Hero Section:** Replicates the main email capture call-to-action (CTA) with layered text and input fields.

💻 Tech Stack
*   HTML5
*   CSS3
